scanf = 사람이 친 값을 변수로 저장시킨다
scanf 사용방법
{
scanf("%변수종류", &변수 이름);
-----  --------   ---------
  1        2          3
1 - scanf를 친다
2 - %변수종류를 친다(%d, %f, %lf, %ld 등)
3 - &변수이름을 친다
***중요! - 변수 만들 땐 int, float, double, stra 등으로 시작하여 변수이름을 적고 ;를 붙인다(처음에 적는 것에 따라 변수 타입이 바뀐다)***
}
프로그래밍에는 =은 절대로 같다는 뜻이 아님, 오른쪽 값을 왼쪽 변수에 저장함(==가 같음)
