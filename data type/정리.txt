10진수 -> 2진수:
  for 10진수가 1이 될때까지:
      10진수 숫자를 2로 나눈다
  return 1을 시작으로 차근차근 나머지를 붙인다 (ex 8 = 1000)
  
%d = 정수형 표현(int)
%ld = 정수형 표현(long)
%c = 문자형 표현(char)
%f = 실수형 표현(float)
%lf = 실수형 표현(double)
%s = 문자열 표현(char *, char [])
%o = 8진수 정수형(int)
%x, %X = 16진수 정수형(int)

변환 문자 = [%d, %ld. %c. %f ''' %x, %X]
